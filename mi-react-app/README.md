# Taller Automotriz - Frontend

Frontend del sitio web de Taller Automotriz, desarrollado con React y Vite.

## Tecnologías

- React 18
- Vite
- CSS3
- HTML5

## Instalación

```bash
git clone https://github.com/Zdm17y4/Frontend-Taller-Automotriz.git
cd mi-react-app
npm install
npm run dev
```

Disponible en `http://localhost:5173`

## Scripts

- `npm run dev` - Servidor de desarrollo
- `npm run build` - Build para producción
- `npm run preview` - Preview de la compilación
